# JavaScript Self-Learning Journey

Welcome to my personal JavaScript learning repository! This space serves as a comprehensive log of my progress, featuring hands-on exercises, foundational UI projects, and independent applications developed as I master JavaScript. 

The repository is structured to separate guided learning from my own independent implementation and problem-solving.

---

## 📂 Repository Structure

The project is organized into distinct modules based on the learning phase and implementation type:

* **`01-getting-started/`** * **Focus:** Core fundamentals and developer environment setup.
    * **Content:** Introductory exercises, DOM manipulation basics, and working with browser DevTools to debug code effectively.
* **`2-ui/`** * **Focus:** Practical user interface elements.
    * **Content:** Core UI components, event handling, dynamic styling, and creating interactive front-end elements.
* **`My Self/`** * **Focus:** **Independent Application & Development.**
    * **Content:** Projects, coding challenges, and logic built entirely from scratch to test my skills, apply theoretical concepts, and solve problems independently without tutorial guidance.

---

## 🚀 Key Technologies & Concepts Covered

* **Language:** JavaScript (ES6+)
* **Web Fundamentals:** HTML5, CSS3, DOM Manipulation
* **Tooling:** Browser Developer Tools (Debugging, Console, Network)
* **Concepts:** Event-driven programming, asynchronous execution, and data management.
